# ASCII Tactical

A not fun local multiplayer tactical shooter in your terminal. Think Counter-Strike but in ASCII art. Pure Node.js, no complicated setup.

---

## SNAPSHOT CHANGELOG

### Latest (v0.2)
- **New visual TUI** — map and HUD side by side in an 80-column frame with box borders
- **Health/armor bars** — `█░` block indicators with color gradient (green → yellow → red)
- **Live scoreboard** — K/D and HP for all players visible in real-time
- **Colored server logs** — timestamps and event labels (JOIN, LEAVE, GAME, MATCH, ERROR)
- **Text overflow fixes** — all messages clamped to frame width, no visual glitches
- **Aiming reticle** — yellow directional line shows where you're aiming; enemies in your line of fire highlight with a yellow background ("locked on")

Have fun!
